# i18n — Çok Dil Altyapısı

> Modül: Bildirim & i18n (08-notification-i18n) | Versiyon: 1.0 | Son güncelleme: 2026-03-17
> Kanonik kaynak: SupportedLocale, TranslationKey, TranslationValue, TenantLocaleConfig, UserLocaleConfig burada tanımlanır.

## Dil Öncelik Sırası

1. Kullanıcı dil ayarı → 2. Tenant default → 3. Sistem default (EN)

## Entity'ler

**SupportedLocale** (`i18n.supported_locale`): code, name, isRtl — sistem tanımlı.

**TranslationKey** (`i18n.translation_key`): keyCode (benzersiz), module, defaultValue (EN), description.

**TranslationValue** (`i18n.translation_value`): translationKeyId FK, locale, value, isOverride, tenantId.

**TenantLocaleConfig** (`i18n.tenant_locale_config`): defaultLocale, supportedLocales JSONB, dateFormat, timeFormat, timezone, currency.

**UserLocaleConfig** (`i18n.user_locale_config`): userId FK, locale, dateFormat, timezone.

## Desteklenen Diller: EN, TR, DE, FR, AR (RTL)
