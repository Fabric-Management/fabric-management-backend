## ‚ö†Ô∏è PROCUREMENT-TIME FIELD OVERRIDES

### Fields Determined at Purchase

**OriginType (ALWAYS at procurement):**

```
Base Fiber: Cotton (originType=UNKNOWN)
                ‚Üì
Purchase from Supplier A ‚Üí originType=DOMESTIC
Purchase from Supplier B ‚Üí originType=IMPORTED
Purchase from Supplier C ‚Üí originType=MIXED (domestic + imported blend)
```

**SustainabilityType (CAN UPGRADE):**

```
Base Fiber: Cotton (sustainabilityType=CONVENTIONAL)
                ‚Üì
Purchase ORGANIC batch ‚Üí sustainabilityType=ORGANIC (upgrade)
Purchase BCI batch ‚Üí sustainabilityType=BETTER_COTTON
Purchase RECYCLED ‚Üí sustainabilityType=RECYCLED
```

**FiberProperty (SPECIFIC to batch):**

```
Base Fiber: Cotton (property=NULL)
                ‚Üì
Purchase Premium batch:
- stapleLength: 35mm (long staple)
- fineness: 1.5 dtex (fine)
- tenacity: 3.0 cN/tex
- color: RawWhite
```
